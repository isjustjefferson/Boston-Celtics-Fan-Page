# Boston Celtics Fan Page 

Projeto desenvolvido durante a disciplina de Desenvolvimento Web 1 no IFPE - Campus Paulista, focado na criação de uma interface temática. 

## Descrição
O projeto consiste em uma fanpage dedicada ao time de basquete Boston Celtics. O objetivo principal foi aplicar conceitos fundamentais de desenvolvimento web, como estruturação semântica, estilização personalizada e layout responsivo.

## Tecnologias Utilizadas
* **HTML5**: Estruturação semântica do conteúdo.
* **CSS3**: Estilização, focando na identidade visual da franquia (verde, branco e dourado).
* **JavaScript**: Apenas para a estruturação de uma tabela.

## O que este projeto demonstra:
* **Atenção aos Detalhes**: Fidelidade à identidade visual de uma marca real.
* **Fundamentos Web**: Domínio das tecnologias que servem de base para qualquer aplicação moderna.
* **Organização**: Estrutura de arquivos clara e código limpo.

## Como visualizar
Você pode acessar por um repositório local:
* Clone o repositório: `git clone https://github.com/isjustjefferson/Boston-Celtics-Fan-Page.git`
* Abra o arquivo `index.html` em seu navegador.

Ou então acessar o link do GitHub Pages:
